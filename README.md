 Student Record System (C)

 Project Description

This project is a simple **Student Record System** developed using the C programming language.
It is a console-based application that allows users to store and display student information.

The system manages basic details such as:

* Student Name
* Roll Number
* Age
* Marks

This project helps beginners understand how to handle structured data in C.

 Features

*  Add student records
*  Display student details
*  Handle multiple students (using arrays)
*  Simple and beginner-friendly
*  Fast execution using C

Technologies Used

* C Programming Language
* Standard Libraries (`stdio.h`)

 Concepts Covered

* Structures (`struct`)
* Arrays
* Functions
* Loops (`for`, `while`)
* Conditional Statements (`if-else`)
* Input/Output using `scanf` and `printf`

 How to Run

1. Compile the program:

gcc student_record.c -o student

2. Run the program:

./student
